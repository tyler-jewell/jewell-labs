# Skills

Hermes **skills** published from this repo. A skill is a directory containing a
`SKILL.md` (plus any helper scripts/resources).

## Install a skill

```sh
hermes skills install tyler-jewell/jewell-labs/skills/<skill-name>
```

## Or tap the whole repo as a skill source

```sh
hermes skills tap add tyler-jewell/jewell-labs
hermes skills search <query>      # then: hermes skills install <name>
```

## Publish a skill here

```sh
hermes skills publish <path-to-skill-dir> --to github --repo tyler-jewell/jewell-labs
```
